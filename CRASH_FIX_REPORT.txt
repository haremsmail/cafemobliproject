ROOT CAUSES IDENTIFIED AND FIXED:
================================================================================

1. ✅ SPLASH ACTIVITY THREADING ISSUE
   Problem: Using bare Thread for UI operations causes race conditions
   Fix: Replaced with Handler/Looper for main thread operations
   File: SplashActivity.java

2. ✅ NULL POINTER EXCEPTIONS
   Problem: No null checks on view initialization
   Fix: Added try-catch and null checks in initViews()
   Files: LoginActivity.java, MainActivity.java

3. ✅ MISSING CLICK LISTENER GUARDS
   Problem: Setting click listeners without null checks
   Fix: Added null checks before setOnClickListener()
   Files: LoginActivity.java, MainActivity.java

4. ✅ ROOM DATABASE MAIN THREAD QUERIES
   Problem: Database queries on main thread without allowMainThreadQueries()
   Fix: Added .allowMainThreadQueries() to Room builder
   File: CafeDatabase.java

5. ✅ UNHANDLED DATABASE EXCEPTIONS
   Problem: Menu item insertion without error handling
   Fix: Wrapped all database operations in try-catch blocks
   File: MainActivity.java
