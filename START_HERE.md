## 🚀 HOW TO RUN (Quick Steps)

### Step 1: Open in Android Studio
```
File → Open → Select projectyear folder
```

### Step 2: Wait for Gradle Sync
```
File → Sync Now (if needed)
```

### Step 3: Select Emulator or Device
```
Tools → Device Manager → Create/Select device
```

### Step 4: Run App
```
Run → Run 'app' (or Shift+F10)
```

### Step 5: Test
```
1. Splash screen shows (3 seconds)
2. Create account (any email/password)
3. Browse menu, add items
4. Checkout to place order
5. View profile to see orders
```
