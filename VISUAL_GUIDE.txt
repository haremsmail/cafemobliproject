## 🆘 TROUBLESHOOTING IN 1 MINUTE

```
❌ PROBLEM: "Build Failed"
✅ SOLUTION:
   1. Build → Clean Project
   2. Build → Rebuild Project
   3. Try running again

❌ PROBLEM: "Device not found"
✅ SOLUTION:
   1. Plug in device with USB
   2. Enable USB debugging on device
   3. Or open Android Emulator in AVD Manager
   4. Wait 30 seconds for device to connect
   5. Try running again

❌ PROBLEM: "App crashes on startup"
✅ SOLUTION:
   1. File → Invalidate Caches → Invalidate & Restart
   2. Wait for Android Studio to restart
   3. Build → Rebuild Project
   4. Run again

❌ PROBLEM: "Still not working?"
✅ SOLUTION:
   1. Close Android Studio completely
   2. Close emulator/disconnect device
   3. Restart Android Studio
   4. Reopen project
   5. Wait 1 minute for indexing
   6. Try building again
```
